# PHP Singularizer

Lightweight utility to convert plural form of words to singular


## Getting Started

### Intall package
```
composer require gadelgado/singularizer
```

### Use
```
<?php

use Gadelgado\Singularizer\Singularizer;

$singular = Singularizer::singularize($plural);
```
